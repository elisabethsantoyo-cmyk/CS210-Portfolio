# CS210-Portfolio
Project Summary

This project is a C++ program called Corner Grocer that analyzes grocery purchase data. The program reads items from an input file and counts how many times each item appears. It then allows the user to search for item frequencies, display all items with their counts, and show a histogram. The goal of the program is to help track purchase patterns and organize data in a simple way.

What I Did Well

I did well in organizing my code and making sure the program worked correctly with file input and output. I also made the menu easy to follow so the user can interact with the program without confusion.

Where I Could Improve

I could improve my code by adding more input validation and making it more efficient. For example, I could handle invalid user inputs better and clean up some repeated code to make it more readable.
